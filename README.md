# 🔗 URL Shortener Project

A professional URL Shortener web application built using:

- Node.js
- Express.js
- SQLite
- HTML
- CSS
- JavaScript

## 🚀 Features

- URL Shortening
- SQLite Database
- URL Validation
- QR Code Generator
- Copy Short URL
- Responsive UI
- Animated Design

## 📦 Installation

```bash
npm install
npm run dev
